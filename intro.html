Introduction to HTML (History)

➾Development

The history of HTML (Hypertext Markup Language) development is marked by significant milestones, standardization efforts, and technological advancements. Here is an overview of the key events and versions in the history of HTML:

  1. HTML (1991): HTML was created by Tim Berners-Lee, a British computer scientist, at CERN (the European Organization for Nuclear Research) as a way to share documents among scientists. The first version of HTML was a simple markup language for linking and structuring text documents.


  2. HTML+ (1993): HTML+ introduced new features like tables and forms, enhancing the capabilities of the language.


  3. HTML 2.0 (1995): The first formal HTML specification was published by the Internet Engineering Task Force (IETF). HTML 2.0 included basic formatting and linking capabilities.


  4. HTML 3.0 (1995): The HTML 3.0 draft specification was introduced, but it was never widely implemented, and its features were later incorporated into subsequent versions.


  5. HTML 3.2 (1997): HTML 3.2, also known as "Wilbur," introduced many new features like tables, image maps, and forms. It provided a more structured and richer way to create web pages.


  6. HTML 4.0 and 4.01 (1997/1999): HTML 4.0 and its subsequent revision, HTML 4.01, introduced cascading style sheets (CSS) support, better forms, and other improvements. It was widely adopted and formed the basis for web development for many years.


  7. XHTML 1.0 (2000): XHTML (Extensible Hypertext Markup Language) was introduced, aiming to bring HTML into compliance with XML (eXtensible Markup Language). XHTML 1.0 followed the rules of XML, making it more strict and well-formed.


  8. HTML5 (2008): HTML5 marked a significant shift in web development. It introduced a wide range of new elements and features, including multimedia support, semantic tags (like `<header>`, `<nav>`, and `<footer`>), the `<canvas>` element, and improved form handling. HTML5 also improved support for multimedia elements such as video and audio.


  9. HTML 5.1 (2016): An incremental update to HTML5, HTML 5.1 added some features and refinements to the language.


  10. HTML 5.2 (2017): Another incremental update to HTML5, HTML 5.2 included new elements, APIs, and features for web development.


  11. HTML Living Standard (Ongoing): Rather than releasing major versions, HTML development now follows a "living standard" approach, meaning it's continuously evolving and being updated as new features and technologies emerge. The specification is maintained by the World Wide Web Consortium (W3C) and the Web Hypertext Application Technology Working Group (WHATWG).


HTML has come a long way from its early beginnings as a simple text markup language to a versatile and feature-rich standard for web development. HTML5, in particular, has significantly expanded the capabilities of web applications, making it easier to create rich and interactive websites and web applications.

➾HTML Version Timeline

HTML 2

  • November 24, 1995
  • HTML 2.0 was published as RFC 1866.

HTML 3

  • January 14, 1997
  • HTML 3.2 was published as a W3C Recommendation.

HTML 4

  • December 18, 1997
  • HTML 4.0 was published as a W3C Recommendation.
  • Initially code-named "Cougar", HTML 4.0 adopted many browser-specific element types and attributes, but also sought to phase out Netscape's visual markup features by marking them as deprecated in favor of style sheets. HTML 4 is an SGML application conforming to ISO 8879-SGML.
  • HTML 4.01 was published as a W3C Recommendation. It offers the same three variations as HTML 4.0 and its last errata were published on May 12, 2001.
  • After HTML 4.01, there were no new versions of HTML for many years, as the development of the parallel, XML-based language XHTML occupied the W3C's HTML Working Group.

HTML 5

  • October 28, 2014
  • HTML5 was published as a W3C Recommendation.
  • HTML 5.1 was published as a W3C Recommendation in November 1, 2016.
  • HTML 5.2 was published as a W3C Recommendation in December 14, 2017.
  • In September 2014, W3C moved HTML5 to Proposed Recommendation.
  • On 28 October 2014, HTML5 was released as a stable W3C Recommendation, meaning the specification process is complete.
